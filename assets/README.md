# Assets

`extension.py` and `scripts/load_warehouse.py` both expect a USD stage at:

```
assets/project_assembly.usd
```

This file (the warehouse + drone scene) is the binary USD asset that ships
with the original Isaac Sim project template and was not included in your
upload, so it is not part of this scaffold. Drop your `project_assembly.usd`
here (or update the path in `extension.py` / `load_warehouse.py`) before
running the simulation.
